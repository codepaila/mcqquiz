AI Settings

Configure the AI provider used by the "Improve with AI" button on question explanations. Works with DeepSeek or any other OpenAI-compatible chat completions API.

🔌 Provider

Connection details for the AI API. The endpoint must accept an OpenAI-compatible /chat/completions request.

📝 Prompt Template

Sent to the AI exactly as written below, with these placeholders filled in automatically: {{question}} {{options}} {{correct_answer}} {{explanation}} — the default prompt asks it to improve the explanation only, but you can edit it to ask for anything else instead.

⚙️ Status

Enable AI assistant
When off, the "Improve with AI" button shows an error instead of calling the API.
